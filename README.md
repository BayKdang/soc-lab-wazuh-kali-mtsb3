# SOC Lab (SIEM)
SOC Lab using Kali Linux, Ubuntu Server and Metasploiatble 3.

## This project simulates a scenario of an attack of a device or machine:

- **Simulate cyberattacks and test security defenses**
- **Kali Linus attack vulnerable Metasploitable 3 using Nmap scan and Hydra for brute force**
- **Implement Real time alerts and investigation**
- **Monitor traffic and analyze logs**
- **Response with proper way (ip block, ssh block)**
- **Document findings for educational and operational use**
  
This serves as a learning and educational purpose.
