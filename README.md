# eNSSTP
Enterprise Networking Simulation &amp; Security Testing Platform project starting soon....

## This project simulates a full enterprise network in a virtual environment to:

- **Design and configure realistic network topologies**
- **Implement VLANs, routing, ACLs, and firewall rules**
- **Simulate cyberattacks and test security defenses**
- **Monitor traffic and analyze logs**
- **Document findings for educational and operational use**
  
It serves as a learning platform, a security testing lab, and a thesis project.
